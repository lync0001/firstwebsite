1. De gebruiker voert een url in in de browser. 2. De browser ontvangt deze url en stuurt onder andere een (http) licenseID request naar de server. 3. De server ontvangt de http request en gaat opzoek naar het gewenste document in de DocumentRoot. 4. De DocumentRoot houdt alle bestanden, en geeft het gewenste bestand vrij aan de server.
5. De server speelt dit bestand (de website) door naar de browser. 6. De browser ontvangt het bestand en voert de code uit. 7. De gebruiker kan vervolgens op eventuele links op de pagina klikken en zo begint het proces opnieuw. 8. En dat houdt Harold blij!
